# 24Meds - Deliveries engine service

##### OnYourWay is a deliveries platform

How to run:

1. in */deliveries* directory:

```bash
chmod 777 build.sh
./build.sh
```

2. In another terminal in */deliveries* directory:

```bash
./mvnw spring-boot:run
```

3. Open webpage in *url*: ``localhost:8080``
